# The COVID Tracking Project

***Total Suggested time: 100 minutes***

Launched in March 2020, the COVID Tracking Project was a volunteer-powered data journalism project to count case and testing information on COVID-19 from every state in country.

In the first two parts of this special podcast from [Reveal](https://revealnews.org/), at the Center for Investigative Reporting, you'll learn more about the origins of the project â€“ and why its founders thought it was necessary to fill in the gaps in data reporting.
